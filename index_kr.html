향후 프로젝트

yujin_ive_lora

yujin_ive_lora는 표준 체크포인트 모델을 수정한 소형 Stable Diffusion 모델인 LoRA 모델을 사용하여 K팝 그룹 IVE의 아이돌 유진의 사진을 생성하여 광범위한 모델 컬렉션을 보유한 사람들에게 어필합니다.

최근 프로젝트

FERD

FERD는 컴퓨터 비전을 사용하여 얼굴 표정을 분석하여 감정을 추론하고 심리학, 인간-컴퓨터 상호 작용, 시장 조사에 적용할 방법을 찾습니다.

완료된 프로젝트

Image Classifier

이미지 분류기는 머신러닝을 사용하여 이미지 인식, 콘텐츠 필터링, 시각적 검색에 적용하여 시각적 특징과 패턴을 분석하여 이미지를 다양한 클래스 또는 카테고리로 정확하게 분류하는 컴퓨터 비전 시스템입니다.
